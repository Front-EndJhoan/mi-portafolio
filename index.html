Proyectos realizados hasta ahora

Landing Page – Calma Mindfulness

Este proyecto consiste en una landing page desarrollada completamente desde cero, utilizando HTML5, CSS3 y JavaScript puro, sin frameworks ni librerías externas. El objetivo fue demostrar dominio sólido de los fundamentos del desarrollo frontend y la capacidad de construir interfaces optimizadas, escalables y bien estructuradas.

La arquitectura del código se diseñó siguiendo buenas prácticas de semántica, accesibilidad y organización modular. En CSS trabajé con metodología estructurada para mantener estilos reutilizables y escalables, incorporando diseño responsive mediante media queries y enfoque mobile-first.

Desde el punto de vista funcional, implementé interactividad dinámica con JavaScript nativo, optimizando el rendimiento y evitando dependencias innecesarias. La página prioriza:

  1. Carga rápida y optimización de recursos
  2. Adaptabilidad total a dispositivos móviles
  3. Buenas prácticas de accesibilidad
  4. Enfoque en conversión y experiencia de usuario

Página Corporativa – Fuente Vieja

Desarrollé la página web para una empresa local de Zaragoza especializada en la producción de aceite de oliva y productos derivados. El proyecto tuvo un enfoque estratégico: transmitir tradición, calidad y profesionalismo mediante una experiencia digital moderna y clara.

La estructura del sitio fue diseñada para facilitar la navegación, resaltar el catálogo de productos y comunicar el proceso de producción. Se priorizó:

  1. Diseño visual alineado con la identidad de marca
  2. Optimización SEO para posicionamiento local
  3. Diseño responsive para todos los dispositivos
  4. Organización clara del contenido corporativo

Trabajé especialmente la jerarquía visual, el uso del color y la presentación del producto para reforzar la percepción de calidad. Además, implementé formularios de contacto optimizados y llamadas a la acción estratégicamente ubicadas para fomentar la captación de clientes.

Página Web en WordPress – Grupo de Mariachis

En este proyecto desarrollé el sitio web para un grupo de mariachis en Colombia utilizando WordPress como CMS. El objetivo principal fue crear una plataforma autogestionable que les permitiera mostrar sus servicios, galería multimedia y facilitar contrataciones.

La solución incluyó:

  1. Personalización avanzada de tema
  2. Adaptación del diseño a la identidad del grupo
  3. Optimización de rendimiento y velocidad
  4. Integración de formularios de contacto y reserva
  5. Estructura SEO básica para mejorar visibilidad

Más allá del desarrollo técnico, el reto fue transmitir energía, tradición y profesionalismo mediante el diseño visual, asegurando al mismo tiempo una experiencia fluida en dispositivos móviles, considerando que gran parte del tráfico proviene de smartphones.
